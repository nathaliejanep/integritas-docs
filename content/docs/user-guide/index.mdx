## Prerequisites

To use the Integritas MiniDapp, you need to run a node on the **Minima blockchain**.

---

## Minima Node Setup

Follow the [Minima Run a Node Guide](https://minima.global/technology/run-a-node) to get started. This guide will walk you through the installation process and ensure your node is fully operational.

---

## Accessing Minima Integritas

Integritas is optimized for use with **Minima Desktop Connect**.

### 1. Open the Minima MiniDapp Hub

After running your node, access the MiniDapp Hub at:

- **Local access**:  
  `https://127.0.0.1:9003` or `https://localhost:9003`

- **Remote access**:  
  `https://YourServerIP:9003`  
  _(Replace `YourServerIP` with the IP address of your remote server.)_

### 2. Authenticate with Minima

Log in using your **Minima credentials** to authenticate your session.

### 3. Open the Integritas MiniDapp

From the MiniDapp Hub, locate and launch the **Integritas** MiniDapp.
