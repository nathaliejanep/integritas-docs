---
title: Analyze Data
description: This section explains how to verify a file using the Integritas Web App
---

### 1. Navigate to the My Uploads Page

Once inside the Integritas Web App, go to the **My Uploads** page.

### 2. Check Status

Click **Check Status** to check if your data is on chain, this can take up to one minute after you Uploaded and Stamped your data.

### 3. Download Proof File

Once the data is on chain, download the proof file you wish to verify.

### 4. Navigate to the Analyze Page

Next, go to the **Analyze** page to to begin the verification process.

### 5. Upload a Proof File

Manually select your file or drag and drop it into the upload area.
