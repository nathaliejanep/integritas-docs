## Managing Files

### Delete a File from the Upload Box

To delete a file from the **Upload** box (under the **Upload Data** or **Analyze** page).

Click the **Delete** icon next to the file.

This action will remove the file from the Upload box.

### Delete a File from My Uploads

Navigate to **My Uploads**, then click the **Delete** button in the right-end column.  
This will permanently remove the file from your stored uploads.

### Delete a File from My Verifications

Navigate to **My Verifications**, then click the **Delete** button in the right-end column.  
This will permanently remove the file from your stored verifications.
