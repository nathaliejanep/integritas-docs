## Prerequisites

To use the Integritas Web App, you need to sign up for an Integritas subscription.

### Getting Started

1. **Visit Integritas**: Go to [https://integritas.minima.global](https://integritas.minima.global)
2. **Sign Up**: Create your account and choose a subscription plan
3. **Access the App**: Use your credentials to log into the Integritas Web App

<Callout
  title='Important'
  type='info'
>
  You must have an active Integritas subscription to use the web application. If
  you don't have a subscription yet, please visit the Integritas website to sign
  up.
</Callout>

---

## Accessing Integritas+

Integritas is optimized for use with **Desktop**. You may experience some display issues on mobile and smaller screens.

There are two ways to access the Integritas application:

**Option 1**:

Log in on [https://integritas.minima.global](https://integritas.minima.global), after logging in you will be redirected to your profile dashboard. Click the link in the sidebar to access the app.
