## Prerequisites

To use the Integritas Web App, you need a Integritas subscription.

Link to Integritas: [https://integritas.minima.global](https://integritas.minima.global)

---

## Accessing Integritas

Integritas is optimized for use with **Desktop**.

### 1. Go to the Integritas app

Link to Integritas Web App: [https://integritas.minima.global/app](https://integritas.minima.global/app)

### 2. Authenticate

Log in using your **Credentials** to authenticate your session.
