## NFT Proof Trace (Optional Feature)

The **NFT Proof Trace** is an optional feature in the API that allows you to request an NFT trace of the verification process.

When enabled:

- It provides an **immutable, on-chain record** of the validation.
- The verification is **linked to a unique NFT**, ensuring transparency and traceability.

## Request Body (Choose one)

| Request Type               | Content-Type                          | Body Type   | Keys                | Description                                                                                   |
| -------------------------- | ------------------------------------- | ----------- | ------------------- | --------------------------------------------------------------------------------------------- |
| {REQUEST_TYPE_FILE_UPLOAD} | <code>{CONTENT_TYPE_MULTIPART}</code> | `form-data` | `file`              | Validate a source file.                                                                       |
| {REQUEST_TYPE_FILE_UPLOAD} | <code>{CONTENT_TYPE_MULTIPART}</code> | `form-data` | `file`              | Validate a `.tar` file. The `.tar` file should contain the source file and a JSON proof file. |
| {REQUEST_TYPE_FILE_UPLOAD} | <code>{CONTENT_TYPE_MULTIPART}</code> | `form-data` | `file`, `jsonproof` | Validate a source file with a JSON proof file.                                                |
