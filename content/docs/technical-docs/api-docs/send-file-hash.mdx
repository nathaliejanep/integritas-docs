---
title: Send File and Get Hash
description: Submit a file to receive its SHA3-256 hash. This is the first step in the stamping process.
---

import {
  API_BASE_URL,
  ENDPOINT_FILE_HASH,
  HTTP_METHOD_POST,
  HEADER_CONTENT_TYPE,
  HEADER_X_API_KEY,
  HEADER_X_REQUEST_ID,
  CONTENT_TYPE_MULTIPART,
  PLACEHOLDER_API_KEY,
  PLACEHOLDER_REQUEST_ID,
  DESC_X_API_KEY,
  DESC_X_REQUEST_ID,
  REQUEST_TYPE_FILE_UPLOAD,
} from '@/lib/constants';

| Attribute       | Details                                                                        |
| --------------- | ------------------------------------------------------------------------------ |
| **Method**      | <code>{HTTP_METHOD_POST}</code>                                                |
| **Description** | This endpoint is used to submit a file. Send the file as formdata in the body. |
| **Endpoint**    | <code>{API_BASE_URL}{ENDPOINT_FILE_HASH}</code>                                |

---

## Request Headers

| Header           | Value                                 | Description                        |
| ---------------- | ------------------------------------- | ---------------------------------- |
| **Content-Type** | <code>{CONTENT_TYPE_MULTIPART}</code> | Specifies the content type as JSON |
| **x-api-key**    | <code>{PLACEHOLDER_API_KEY}</code>    | {DESC_X_API_KEY}                   |
| **x-request-id** | <code>{PLACEHOLDER_REQUEST_ID}</code> | {DESC_X_REQUEST_ID}                |

---

## Request Body

| Request Type               | Content-Type                          | Body Type   | Keys   | Description |
| -------------------------- | ------------------------------------- | ----------- | ------ | ----------- |
| {REQUEST_TYPE_FILE_UPLOAD} | <code>{CONTENT_TYPE_MULTIPART}</code> | `form-data` | `file` | Send file   |

## API Response

The response from the API contains the hash of the file.

### Example Response

```json
{
  "apiVersion": 1,
  "requestId": "postman-req-id",
  "status": "success",
  "statusCode": 200,
  "message": "Hash computed",
  "timestamp": "2025-06-11T08:12:42.654Z",
  "data": {
    "hash": "C74C4DDE95A49EAFDBA139568E9955C65E017B55662132FA824AF58E6E782427",
    "filename": "test.docx",
    "mimeType": "application/vnd.openxmlformats-officedocument.wordprocessingml.document"
  }
}
```
